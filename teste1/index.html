Jogo da Velha
Sobre o jogo.

O jogo da velha é um jogo de estratégia para dois jogadores em um tabuleiro 3x3. Os jogadores alternam entre marcar espaços com "X" e "O" com o objetivo de formar uma linha horizontal, vertical ou diagonal. O primeiro jogador a completar uma linha vence. Se o tabuleiro ficar completamente preenchido sem linhas completas, o jogo termina em empate.

Campo Minado
Sobre o jogo.

O campo minado é um jogo de lógica onde o objetivo é descobrir todas as minas no tabuleiro. O jogador pode clicar em qualquer quadrado para revelar o que está por baixo. Se o quadrado revelado contiver uma mina, o jogo termina. Se o quadrado não contiver uma mina, o jogador pode continuar a revelar quadrados. O jogo termina quando todas as minas são descobertas ou quando o jogador revela um quadrado com uma mina.

Adivinhação
Sobre o jogo.

O jogo de adivinhação é um jogo onde o jogador tenta adivinhar um número entre 1 e 100. O jogo informa se o número é maior ou menor que o número escolhido. O jogador tem um número limitado de tentativas para adivinhar o número correto. Se o jogador adivinhar o número correto, o jogo termina e o jogador vence. Se o jogador não adivinhar o número correto dentro do número de tentativas, o jogo termina e o jogador perde.
